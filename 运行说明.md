# 🚀 项目运行说明

> ⚠️ **请先配置环境变量**：打开项目根目录下的 `.env` 文件，修改其中的 `API_KEY` 为你的实际密钥，否则后端无法正常调用接口。

---

## 📦 一、安装依赖

使用以下命令安装所需 Python 包：

```bash
pip install openai
pip install dotenv
pip install flask
pip install CORS
```

> 💡 **推荐：** 使用清华镜像源可大幅提升下载速度
>
> ```bash
> pip install -i https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple <包名>
> ```

---

## ⚙️ 二、运行后端 API

```bash
python AI_api.py
```

---

## 🌐 三、访问页面

启动后，在浏览器中打开：

```
http://localhost:5200/chat
```

---

> ✅ 按顺序完成以上三步即可正常使用本项目。
