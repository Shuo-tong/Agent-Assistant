# 🚀 项目运行说明

> ⚠️ **请先配置环境变量**：本项目需要你自行创建 `.env` 文件（参考 `.env.example`），填入你的实际 API 密钥，否则后端无法正常调用接口。

---

## 🔑 一、配置环境变量

项目根目录下创建一个 `.env` 文件，参考以下格式填写你的 API 密钥：

```env
# 通义千问（DashScope）
DASHSCOPE_API_KEY = 你的 DashScope API Key
DASHSCOPE_BASE_URL = https://dashscope.aliyuncs.com/compatible-mode/v1

# DeepSeek
DEEPSEEK_API_KEY = 你的 DeepSeek API Key
DEEEPSEEK_BASE_URL = https://api.deepseek.com
```

> 你也可以直接复制 `.env.example` 为 `.env`，然后填入密钥即可。

## 📦 二、安装依赖

使用以下命令安装所需 Python 包：

```bash
pip install openai
pip install dotenv
pip install flask
pip install CORS
```

> 💡 **推荐：** 使用清华镜像源可大幅提升下载速度
>
> ```bash
> pip install -i https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple <包名>
> ```

---

## ⚙️ 三、运行后端 API

```bash
python AI_api.py
```

---

## 🌐 四、访问页面

启动后，在浏览器中打开：

```
http://localhost:5200/chat
```

---

> ✅ 按顺序完成以上三步即可正常使用本项目。
